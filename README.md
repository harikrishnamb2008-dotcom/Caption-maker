# AI Video Caption Generator

An AI-powered tool for generating and styling video captions with high customization.

## Tech Stack
- **Frontend:** React Native (Expo)
- **Backend/AI:** Python/Node.js with OpenAI Whisper & FFmpeg

## Project Goals
- Automate transcription with high accuracy.
- Allow users to customize font, color, and animation presets.
- Export videos with "burned-in" captions.
